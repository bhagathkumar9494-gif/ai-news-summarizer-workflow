# AI News Summarizer

## Overview
This n8n workflow automatically collects AI-related news from multiple RSS feeds, summarizes articles using Google Gemini AI, and sends a daily email report.

## Features
- Scheduled execution
- Multiple RSS sources
- Data aggregation
- AI-powered summarization
- Automated email delivery

## Workflow

Schedule Trigger
↓
RSS Feeds
↓
Merge
↓
Aggregate Data
↓
Gemini Summarization
↓
Email Delivery

## Technologies
- n8n
- Google Gemini
- RSS Feeds
- Gmail
